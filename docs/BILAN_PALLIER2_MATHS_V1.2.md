# (4) RÈGLES DE SCORING v1.2

## 4.1 ReadinessScore

```
Pour chaque domaine D actif (≥2 compétences évaluées):
  Score_D = (moyenne(mastery) / 4) × 100

ReadinessScore = Σ(weight_D × Score_D) / Σ(weights domaines actifs)

Pondérations:
  - Analyse: 30%
  - Algèbre: 25%
  - Géométrie: 20%
  - Probabilités: 15%
  - Python: 10%
```

## 4.2 RiskIndex

```
Performance = 0.40 × (scoreMiniTest/6×100) 
            + 0.20 × (completedInTime ? 100 : 40)
            + 0.25 × (moyenne(redaction, justifications)/4×100)
            + 0.15 × (stress/4×100)

RiskIndex = 100 - Performance
```

## 4.3 Décision

```
SI ReadinessScore >= 60 ET RiskIndex <= 55
  → Pallier2_confirmed
SINON SI ReadinessScore >= 48 ET RiskIndex <= 70
  → Pallier2_conditional
SINON
  → Pallier1_recommended
```

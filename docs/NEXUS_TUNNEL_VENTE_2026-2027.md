## Tunnel de Vente Étape par Étape

### Phase 1 : Premier Contact (WhatsApp/Téléphone)

**Objectif :** Collecter le profil complet

**Questions obligatoires (checklist) :**
- [ ] Niveau (3ème / Seconde / Première / Terminale)
- [ ] Statut (scolarisé / candidat libre / double cursus)
- [ ] Spécialités (si Première/Terminale)
- [ ] Établissement actuel
- [ ] Objectif (consolidation / mention / sélectif / rattrapage)
- [ ] Budget (confortable / standard / serré)
- [ ] Modalité candidat libre (A/B/indecis) si applicable
- [ ] Localisation (Tunis / hors Tunis)
- [ ] Urgence (cherche pour maintenant / septembre)

**Action :** Saisir dans l'outil `nexus_assistante_devis_v2.html` → Générer recommandation

**Checklist à valider :**
- [ ] Le parent a bien compris le positionnement (différence soutien scolaire classique)
- [ ] Les inclusions sont claires (bulletins, bacs blancs, plateforme)
- [ ] L'échéancier est acceptable
- [ ] Les conditions générales sont évoquées (acompte non remboursable sauf si Nexus annule)
- [ ] Une date butoir pour la décision est fixée
- [ ] Une objection éventuelle est levée

**Phrase de fermeture :**
```
Parfait, je résume ce qu'on a dit :
• Offre [NOM] pour [PRÉNOM] : [MONTANT] TND
• Échéancier : [DÉTAIL]
• Pour bloquer la place : acompte de [X] TND

Dès réception de l'acompte, on bloque la place dans le groupe de 5,
```

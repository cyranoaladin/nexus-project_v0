# Parcours 360 V0 Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Construire une chaîne locale qui valide un dossier Parcours 360, génère les quatre bilans en deux phases, impose une relecture nominative et publie uniquement des PDF audités.

**Architecture:** Un paquet Python isolé sous `tools/parcours360/` sépare domaine, génération, relecture, rendu et audit. Les faits restent dans `source.json`; chaque phase produit un brouillon et une validation liés par hashes avant une publication atomique. Le mode manuel est déterministe et l’adaptateur Claude facultatif implémente le même contrat.

**Tech Stack:** Python 3.12, `jsonschema`, Jinja2, WeasyPrint, Matplotlib/SVG, PyPDF, BeautifulSoup, PyYAML, pytest, Poppler, qpdf, ImageMagick, API Messages Anthropic facultative.

**Spec:** `docs/superpowers/specs/2026-07-19-parcours-360-v0-design.md`

**Working-tree rule:** les modifications préexistantes dans `scripts/pre-rentree/`, `generated/` et `__tests__/campaigns/` appartiennent à un autre chantier. Ne pas les modifier, les indexer ni les inclure dans les commits Parcours 360.

- [ ] **Step 3: Implement safe UTF-8 JSON loading and the closed Draft 2020-12 schema**

Implement:

```python
def load_json(path: Path) -> dict[str, Any]:
    try:
        value = json.loads(path.read_text(encoding="utf-8"))
    except (OSError, UnicodeError, json.JSONDecodeError) as error:
        raise ValidationFailure(f"JSON illisible: {path}") from error
    if not isinstance(value, dict):
        raise ValidationFailure(f"Objet JSON attendu: {path}")
    return value


def atomic_write_json(path: Path, value: Mapping[str, Any]) -> None:
    path.parent.mkdir(parents=True, exist_ok=True)
    payload = json.dumps(value, ensure_ascii=False, indent=2, sort_keys=True) + "\n"
    temporary = path.with_name(f".{path.name}.tmp-{os.getpid()}")
    temporary.write_text(payload, encoding="utf-8")
    with temporary.open("rb") as handle:
        os.fsync(handle.fileno())
    os.replace(temporary, path)


def load_schema(name: str) -> dict[str, Any]:
    resource = resources.files("parcours360.domain.schemas").joinpath(name)
    return json.loads(resource.read_text(encoding="utf-8"))
```

`atomic_write_json` writes UTF-8 to a sibling temporary file, flushes and `fsync`s it, then uses `os.replace`. `source.schema.json` must encode every field, enum, bound, uniqueness rule and `additionalProperties: false` requirement from spec section 5. - [ ] **Step 3: Implement the RFC 8785-compatible restricted canonicalizer**

Reject floats and non-string object keys. The accepted source subset is null, bool, integer, string, list and mapping with ASCII schema keys. Serialize with UTF-8, sorted keys, no insignificant whitespace and `ensure_ascii=False`; document why this is RFC 8785-compatible for the restricted data model.

```python
def canonical_bytes(value: JsonValue) -> bytes:
    def reject_unsupported(item: JsonValue) -> None:
        if isinstance(item, float):
            raise TypeError("Les nombres flottants sont interdits")
        if isinstance(item, dict):
            if any(not isinstance(key, str) or not key.isascii() for key in item):
                raise TypeError("Les clés JSON doivent être des chaînes ASCII")
            for nested in item.values():
                reject_unsupported(nested)
        elif isinstance(item, list):
            for nested in item:
                reject_unsupported(nested)

    reject_unsupported(value)
    return json.dumps(
        value, ensure_ascii=False, sort_keys=True, separators=(",", ":"), allow_nan=False,
    ).encode("utf-8")


def sha256_json(value: JsonValue) -> str:
    return hashlib.sha256(canonical_bytes(value)).hexdigest()
```

Add a golden test whose expected SHA-256 is computed once from a fixed literal and committed as a constant, plus a test that reordered keys produce the same hash.

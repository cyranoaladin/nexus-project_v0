# (4) RÈGLES DE SCORING v1.1

## 4.1 ReadinessScore (Readiness pour Pallier 2)

### Pondérations par domaine

| Domaine | Pondération |
|---------|-------------|
| Analyse | 30% |
| Algèbre | 25% |
| Géométrie | 20% |
| Probabilités | 15% |
| Algorithmique | 10% |

### Formule

```
Pour chaque domaine D:
  Si (nb_compétences_étudiées_D / total_compétences_D) >= 50% ALORS
    Score_D = (Σ mastery_i) / (nb_étudiées × 4) × 100
  SINON
    Score_D = null (domaine ignoré pour ce calcul)

ReadinessScore = Σ (Score_D × pondération_D) / Σ pondérations_activées

Si > 50% des domaines sont null, appliquer pénalité de -5 points
```

### Interprétation

| Score | Niveau | Action |
|-------|--------|--------|
| 0-47 | Insuffisant | Pallier 1 recommandé |
| 48-59 | Fragile | Pallier 2 sous condition |
| 60-100 | Adéquat | Pallier 2 confirmé |

## 4.2 RiskIndex (Risque Épreurologique 2026)

### Pondérations

| Facteur | Pondération |
|---------|-------------|
| Automatismes | 40% |
| Temps | 20% |
| Rédaction | 25% |
| Stress | 15% |

### Formule

```
RiskIndex = (
  (4 - automatisms_score) / 4 × 40 +
  (4 - time_score) / 4 × 20 +
  (4 - redaction_score) / 4 × 25 +
  (4 - stress_score) / 4 × 15
) × 100 / (40 + 20 + 25 + 15) × 100
```

### Interprétation

| RiskIndex | Niveau |
|-----------|--------|
| 0-55 | Risque faible |
| 56-70 | Risque modéré |
| 71-100 | Risque élevé |

## 4.4 Gestion des null

1. **Exclusion** : Compétences avec status = not_studied → mastery = null → exclues du calcul
2. **Pénalité** : Si > 50% domaines null → -5 points
3. **Incomplet** : Si données manquantes → flag "incomplete" + scores calculés avec disponibles

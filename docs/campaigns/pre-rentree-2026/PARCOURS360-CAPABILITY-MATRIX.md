## Règle de publication

Le générateur n’accepte comme promesse ferme que les libellés dont `publiclyCommitted` vaut `true` dans `parent-guide.fr.json`. Les quatre documents personnalisés, les douze tests disciplinaires, leur scoring et leur diffusion nominative sont volontairement absents du Guide Parents.

La matrice n’est ni une approbation propriétaire du paquet final, ni une validation juridique, ni une validation de confidentialité.

# ADR-0001 — Coexistence Next.js (App Router) + FastAPI (schema Postgres dédié)
Conserver Next.js (App Router, Prisma/NextAuth) sur `public`. Implémenter nouvelles features dans FastAPI avec tables dans `nexus_app`. Dev: rewrite '/pyapi' → FastAPI; Prod: reverse proxy. CI: jobs Node+Python+Compose E2E.
